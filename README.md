# CurrencyX — Currency Converter

A modern, real-time currency converter built with React and Tailwind CSS.

## Features

- Real-time exchange rates via [ExchangeRate API](https://www.exchangerate-api.com/)
- 150+ currencies with full country names
- Swap currencies with one click
- Dark glass-morphism UI
- Responsive design

## Getting Started

```bash
# Clone the repo
git clone https://github.com/ans56alpha-droid/Project.git
cd Project

# Install dependencies
npm install

# Set up environment variables
cp .env.example .env
# Add your API key from https://www.exchangerate-api.com/ to .env

# Start dev server
npm run dev
```

## Build

```bash
npm run build
npm run preview
```

## Tech Stack

- React 19
- Vite
- Tailwind CSS v4
- ExchangeRate API
